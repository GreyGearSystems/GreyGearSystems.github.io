Things to Consider

In respect to feedback about the FPS Integration Tool, we recommend posting a review on the tool's Unity Asset Store page, which will be visable to other users. We can also respond to this publicly to provide you with suggestions on resolving issues with the tool, if you report any.

If problems with the tool are particularly complicated, emailing us about it would be more adequate.

Feedback and questions not directly relating to the tool should also be communicated via email.

Although we will respond as quickly as possible to emails, ones that are irrelevent, spammed or abusive will not be replyed to.

For reasons relating to Data Protection, we encourage you to exclude unneeded identifiable information when communicating to us.
